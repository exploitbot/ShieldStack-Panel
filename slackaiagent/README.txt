# Slack AI Agent Bot - Web Documentation

This directory contains the complete web-based documentation for the Slack AI Agent Bot.

## Access Documentation

Open in web browser:
- http://your-server-ip/slackaiagent/
- Or locally: file:///var/www/html/slackaiagent/index.html

## Available Pages

- index.html - Main documentation hub
- quick-start.html - 30-second getting started guide  
- commands.html - Complete command reference (40+ commands)
- troubleshooting.html - Solutions for common issues
- slash-commands.html - /agent slash command setup
- technical.html - Architecture and technical details
- api-links.html - Direct links to Slack app settings
