Essential Commands

Message What it does
pull [N] Tail the agent log (default 80 lines; max 400)
show [N] Alias of pull that defaults to 50 lines
status Check if the screen session/log exist
command help Runs /help inside the agent session (any text works)
logpath on/off Reveal or hide the log directory in bridge replies
help Show this workflow directly in Slack

Slash Commands

Command Description
/agent status Is the bridge process running?
/agent restart Restart the bridge (not the screen session)
/agent stop Send Ctrl+C to the agent if it is stuck
/agent recent Show the PTY buffer captured by the bridge

üí° Tips

  • The bridge now sends a true Enter keypress, so every prompt is actually submitted.
  • Wait at least the hint shown by the bot (default 1.5s) before running pull/show.
  • Need the log path? Run logpath on (and logpath off to hide it again).
  • Need to see the agent live? screen -r slack-claude, then Ctrl+A ‚Üí D to detach.
  • Trim huge logs with truncate -s 0 logs/claude_screen.log when the agent is idle.
