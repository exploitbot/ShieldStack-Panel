Command What It Does
./init.sh Interactive menu
./init.sh start Full setup (use after reboot)
./init.sh test Check if everything is working
./init.sh logs View recent logs
./init.sh attach Attach to the agent screen (debug mode)
./init.sh stop Stop everything

üí¨ SLACK COMMANDS

Command What It Does
pull Get last 80 lines from the agent log
pull 200 Get last 200 lines
show 50 Same as pull but defaults to 50 lines
status Check system health
command help Send `/help` inside the agent (any text works)
command connectvpn Run VPN helper (also autoconnectvpn, stopvpn)
help Show command list
/agent status Service status
/agent restart Restart bridge

üîß TROUBLESHOOTING

Problem Solution

Old/stale output ./init.sh start
Bot not responding ./init.sh test
Service crashed ./init.sh start
